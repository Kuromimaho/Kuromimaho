# Adam Personal Dashboard

A modern personal dashboard built with **HTML**, **CSS**, and **Vanilla JavaScript**, designed for deployment on **GitHub Pages** with a custom domain.

## ✨ Features

- ⚡ Lightweight and fast
- 📱 Fully responsive
- 🌙 Dark / Light mode
- ⌨️ Typing animation
- 🕒 Live clock
- 🎨 Animated background particles
- 🚀 Smooth scrolling
- 📂 Dynamic website cards from JSON
- 🔝 Scroll to top button
- 🌐 GitHub Pages ready
- 🔗 Custom domain support

## 🚀 Deployment

1. Push this project to GitHub.
2. Open **Settings → Pages**.
3. Select the **main** branch.
4. Save the settings.
5. (Optional) Configure your custom domain:
   - `kuromimaho.web.id`
